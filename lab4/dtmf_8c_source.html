6 //
7 // Compute Goertzel Filter Coefficients for
8 // Frequency Detectors for the following frequencies:
9 //
10 // Gtap structure Fgoertzel
11 // -----------------------------
12 // t697 697 Hz
13 // t770 770 Hz
14 // t852 852 Hz
15 // t1209 1209 Hz
16 // t1336 1336 Hz
17 // t1477 1477 Hz
18 //
19 // A Goertzel filter coefficient is defined by
20 //
21 // c = 2*cos(2*pi*v)
22 //
23 // with v equal to Fgoertzel / Fsample.
24 // For example, if Fgoertzel = 1000Hz and Fsample = 8000Hz
25 // then c = 2 * cos (2*pi*1/8) = 1.4142
26 // Be sure to compute cos() in radians
27 //
28 // This results in the following filter coefficients
29 //
30 // Gtap structure Fgoertzel Coefficient
31 // ----------------------------------------------
32 // t697 697 Hz 0.8538689 * 2
33 // t770 770 Hz 0.8226405 * 2
34 // t852 852 Hz 0.7843435 * 2
35 // t1209 1209 Hz 0.5820520 * 2
36 // t1336 1336 Hz 0.4981851 * 2
37 // t1477 1477 Hz 0.3993092 * 2
38 //
39 // To encode the Goertzel filter coefficient into Gtap,
40 // you have to convert the floating point value of c
41 // into a fixed point value.
42 //
43 // For a fixed point representation <WLEN,FRACBITS>,
44 // The resulting Gtap coef value is
45 //
46 // coef = (int) (c * (1 << FRACBITS))
47 //
48 // Make sure that you do NOT use double or float in your program.
49 // Use ONLY integer types (signed or unsigned).

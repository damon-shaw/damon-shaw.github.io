void DTMFFindSignificant ( bool *  sign_tones)

Determines which of six frequencies has a significant signal strength and marks them as significant in an array of booleans of predefined structure.

Parameters
sign_tonesA pointer to the boolean array (of length 6); each boolean represents if the signal strength of the associated frequency is significant; this representation must be of the form [697, 770, 852, 1209, 1336, 1477].
